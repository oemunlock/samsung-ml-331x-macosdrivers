Job Setting Dialog

Job Setting dialog allows you to select how to print or save the printing file by using the optional HDD. Select the desired one of the following settings:


Normal

Select this setting for normal printing without storing your documents in the HDD. Check Store to HDD check box to save your document in the hard disk after printing. The saved document can be printed at any time you want from the printer's control panel as well as on your PC.

Confidential

Select this setting to send your private or confidential document with a specified password to the hard disk so that you can prevent the other users from accessing it. To print the document, you must enter the password. After printing, the document is cleared from the hard disk. If you want to save the file in the hard disk after printing, check Store to HDD check box.

Store

Select this setting to save a document in the hard disk without printing. When you print the saved document using the control panel, you must enter the password. Use this setting when you store a private or a confidential document. Check Store to Public HDD to save a document in the hard disk without printing. The saved document using this setting can be printed by any users without entering the password. Use this setting when you store public documents.

Spool

This option can be useful to handle large amount of data. If you select this setting, the print spools the document into the hard disk and then prints it from the hard disk queue, decreasing the computer's work load.

Print Schedule

Select this setting to print the document at a specified time. If you set the time to 11 PM when the printer is hardly used, the printer stores the document in the hard disk and prints it at 11 PM.



The Job Properties options are available as below.

User ID

Enter the user's ID. It can be used when you print the file at a later time or when find the saved file using the printer's control panel.

Job Name

Enter the name of the print job. It can be used when you print the file at a later time or when find the saved file using the printer's control panel.

Password

Enter the password in which will be used for Confidential setting. This setting is available only when you select Confidential or Store without checking Store to Public HDD. If you set the password a specific file, you must enter the password to print or delete the file.

Job Name

Select this setting to set the time when the printer begins printing. You can use this setting only if you select Delay.
