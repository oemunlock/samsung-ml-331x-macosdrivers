[Las opciones de configuración pueden variar en función de las impresoras y la versión de Mac OS.]

Cuadro Configuración del trabajo

El cuadro de diálogo Configuración del trabajo permite seleccionar cómo imprimir o guardar el archivo de impresión utilizando la unidad de disco duro opcional. Seleccione el ajuste que desee entre los siguientes:


Normal

Selecciónelo para imprimir normalmente, sin guardar los documentos en el disco duro. Marque la casilla Guardar en disco duro para guardar el documento en el disco duro después de imprimirlo. El documento guardado puede imprimirse en cualquier momento desde el panel de control de la impresora o desde el PC.

Confidencial

Seleccione este ajuste para enviar sus documentos confidenciales o privados al disco duro con una contraseña, a fin de evitar que otros usuarios puedan acceder a ellos. Para imprimir el documento, deberá introducir la contraseña. Después de imprimirse, el documento se borra del disco duro. Si desea guardar el archivo en el disco duro una vez impreso, marque la casilla Guardar en disco duro.

Guardar

Seleccione este ajuste para guardar un documento en el disco duro sin imprimirlo. Cuando imprima el documento utilizando el panel de control, deberá introducir la contraseña. Utilice este ajuste cuando guarde un documento confidencial o privado. Marque la casilla Guardar en disco duro público para guardar un documento en el disco duro sin imprimirlo. Un documento guardado utilizando este ajuste puede imprimirlo cualquier usuario sin necesidad de introducir la contraseña. Utilice este ajuste para guardar documentos públicos.

Spool

Esta opción puede resultar útil para gestionar grandes cantidades de datos. Si selecciona este ajuste, el documento se envía a la cola del disco duro y se imprime desde allí, reduciendo así la carga de trabajo del ordenador.

Program. de impresión

Seleccione este ajuste para imprimir el documento a la hora especificada. Si se ajusta la hora a las 23:00, cuando la impresora apenas se utiliza, el documento se guarda en el disco duro y se imprime a esa hora.



Las opciones de Propiedades de la tarea disponibles son las siguientes.

Usuario

Introduzca el identificador del usuario. Puede utilizarse cuando se imprime el archivo a una hora posterior o para encontrar el archivo guardado utilizando el panel de control de la impresora.

Tarea

Introduzca el nombre de la tarea de impresión. Puede utilizarse cuando se imprime el archivo a una hora posterior o para encontrar el archivo guardado utilizando el panel de control de la impresora.

Contraseña

Introduzca la contraseña que se utilizará para el ajuste Confidencial. Esta opción sólo está disponible cuando se selecciona Confidencial o Guardar sin marcar la casilla Guardar en disco duro público. Si define una contraseña para un archivo, deberá introducirla para poderlo imprimir o borrar.

Imprimir a las

Seleccione este ajuste para especificar la hora a la que se iniciará la impresión. Este ajuste sólo puede utilizarse si se selecciona Retraso.
