[Les options de configuration peuvent varier en fonction des imprimantes et de la version de Mac OS utilisée.]

Boîte de dialogue Réglage de tâche

La boîte de dialogue Réglage de tâche vous permet de choisir la façon d'imprimer ou d'enregistrer le fichier d'impression via le lecteur de disque dur optionnel. Vous pouvez sélectionner l'une des options suivantes :


Normal

Sélectionnez cette option pour les impressions normales sans stockage de documents dans le lecteur de disque dur. Cochez la case Enregistrer dans le lecteur de disque dur pour enregistrer votre document sur le disque dur après l'impression. Le document enregistré peut être imprimé à tout moment depuis le panneau de commande et votre ordinateur.

Confidentiel

Sélectionnez cette option pour envoyer un document privé ou confidentiel protégé par un mot de passe spécifique sur le disque dur afin d'empêcher les autres utilisateurs d'y accéder. Pour imprimer le document, saisissez le mot de passe. Après avoir été imprimé, le document est supprimé du disque dur. Si vous souhaitez enregistrer le fichier sur le disque dur après avoir effectué une impression, cochez la case Enregistrer dans le lecteur de disque dur.

Enregistrer

Sélectionnez cette option pour enregistrer un document sur le disque dur sans l'imprimer. Pour imprimer le document enregistré à l'aide du panneau de commande, saisissez le mot de passe. Cette option vous permet d'enregistrer un document privé ou confidentiel. Cochez la case Enregistrer dans le lecteur de disque dur public pour enregistrer un document sur le disque dur sans l'imprimer. Le document enregistré à l'aide de cette option n'est pas protégé par un mot de passe et peut être imprimé par tous les utilisateurs. Cette option vous permet d'enregistrer des documents publics.

Désynchronisation

Cette option s'avère utile pour le traitement de fichiers volumineux. Si vous sélectionnez cette option, l'imprimante mémorise le document dans une file d'attente sur le disque dur, puis l'imprime depuis cette file d'attente, réduisant ainsi la charge de travail de l'ordinateur.

Planificat impression

Sélectionnez cette option pour imprimer le document à une heure donnée. Si vous réglez l'heure sur 23h00, lorsque l'imprimante n'est presque pas utilisée, l'imprimante enregistre le document sur le disque dur et l'imprime à 23h00.



Les options de propriétés de travail sont les suivantes :

Utilisateur

Saisissez l'ID de l'utilisateur. Cette option sert à imprimer le fichier à une heure plus tardive ou à chercher un fichier enregistré à l'aide du panneau de commande de l'imprimante.

Nom travail

Saisissez le nom du travail d'impression. Cette option sert à imprimer le fichier à une heure plus tardive ou à chercher un fichier enregistré à l'aide du panneau de commande de l'imprimante.

Mot de passe

Saisissez le mot de passe qui sera utilisé dans l'option Confidentiel. Cette option est uniquement disponible si vous sélectionnez Confidentiel ou Enregistrer sans cocher la case Enregistrer dans le lecteur de disque dur public. Si vous définissez un mot de passe pour un fichier, vous devez saisir celui-ci pour l'imprimer ou le supprimer.

Imprimer à

Cette option vous permet de régler l'heure à laquelle l'imprimante doit commencer à imprimer. Cette option est uniquement disponible si vous sélectionnez Différer.
