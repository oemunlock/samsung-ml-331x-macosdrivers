[Les options de configuration peuvent varier en fonction des imprimantes et de la version de Mac OS utilisée.]

Onglet Imprimante

Les options suivantes dépendent des accessoires ou menus disponibles pour votre imprimante.


Mode économie d'encre

L'option Mode économie d'encre vous permet de réduire la quantité d'encre utilisée lorsque vous imprimez le brouillon d'un document. Sélectionnez l'option Désactivé pour imprimer normalement et Activé pour réduire la quantité d'encre.


Correction pour zones de haute altitude

Cochez la case Correction en altitude pour régler l’encre correctement si vous travaillez dans un site situé en altitude.

Réimpression après bourrage

Réimpression après bourrage vous permet de réimprimer, si vous le souhaitez, la dernière page après avoir retiré le papier resté bloqué dans l'imprimante à la suite d'un bourrage.

Economie d’énergie

La case à cocher Economie d'énergie permet d'activer le mode veille de l'imprimante.
