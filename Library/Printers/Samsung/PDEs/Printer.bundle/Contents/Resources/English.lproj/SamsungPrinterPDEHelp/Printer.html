Printer Settings page

The following option depends upon the accessories or item available for your printer.


Toner Save Mode

The Toner Save Mode setting allows the user to save printing toner when printing a draft version of a document. Select Off to print in normal mode and select On to print with less toner applied to the page.

High Altitude Correction

Check High Altitude Correction box if you are located in a high altitude area for adjusting the toner.

Reprint When Jammed

Reprint When Jammed allows users to choose whether you want the printer to reprint the last page printed after you have cleared a jam.

Power Save

The Power Save check-box enables printer to enter sleep mode.
