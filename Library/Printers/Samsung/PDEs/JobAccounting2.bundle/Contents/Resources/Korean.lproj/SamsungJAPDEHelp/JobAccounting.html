작업 계정

작업 계정은 사용자가 출력하는 문서 각각에 대하여 사용자명과 계정인증을 통해서 출력가능하도록 합니다.


사용자 권한

사용자가 해당 옵션을 선택하면, 사용자에게 할당된 권한 설정에 따라서 인쇄 작업에 대한 사용 가능 여부가 결정됩니다.


그룹 권한

사용자가 해당 옵션을 선택하면, 사용자가 속한 그룹에서 할당된 권한 설정에 따라서 인쇄 작업에 대한 사용 가능 여부가 결정됩니다.


작업 계정 비밀번호 암호화

작업 계정 비밀번호 암호화 기능을 선택하면 작업 계정 비밀 번호를 암호화하여 전송합니다.

작업 계정 비밀번호 암호화는 프린터 "정보" 의 "드라이버" 탭에서 설정할 수 있습니다.
