Mode d'impression

Le mode d'impression vous permet de sélectionner la manière d'imprimer ou de sauvegarder le fichier d'impression en utilisant la mémoire (mémoire de masse, mémoire de l'imprimante, mémoire flash interne, disque RAM). Sélectionnez l'un des réglages suivants :


Normal
Sélectionnez ce réglage pour une impression normale sans mémorisation de vos documents dans la mémoire de l'imprimante.

Confidentiel
Sélectionnez ce réglage pour envoyer votre document privé ou confidentiel avec un mot de passe spécifique à la mémoire de l'imprimante de manière à en interdire l'accès à des tiers. Pour imprimer le document, vous devez entrer un mot de passe. Après l'impression, le document est effacé de la mémoire de l'imprimante.

Mémoriser
Sélectionnez ce réglage pour sauvegarder un document sur le disque dur sans l'imprimer. Lorsque vous désirez sauvegarder votre document dans une boîte à documents privée, vous devez entrer un mot de passe. Utilisez ce réglage avec une boîte à documents privée lorsque vous mémorisez un document privé ou confidentiel.

Mémoriser et imprimer
Sélectionnez ce réglage pour sauvegarder un document sur le disque dur et l'imprimer. Lorsque vous désirez sauvegarder votre document dans une boîte à documents privée, vous devez entrer un mot de passe. Utilisez ce réglage avec une boîte à documents privée lorsque vous mémorisez un document privé ou confidentiel.

Programme d'impression
Sélectionnez ce réglage pour imprimer le document à un moment spécifié. Si vous réglez l'heure sur 23 heures lorsque l'imprimante est fortement utilisée, l'imprimante mémorise le document dans la mémoire de l'imprimante et l'imprime à 23 heures.

Les options Propriétés de tâche suivantes sont disponibles.

Nom de la tâche
Entrez le nom de la tâche d'impression. Il peut être utilisé lorsque vous imprimez le fichier ultérieurement ou lorsque vous cherchez le fichier sauvegardé par le biais du panneau de commande de l'imprimante.

ID utilisateur
Entrez l'identifiant utilisateur. Il peut être utilisé lorsque vous imprimez le fichier ultérieurement ou lorsque vous cherchez le fichier sauvegardé par le biais du panneau de commande de l'imprimante.

Mot de passe
Entrez le mot de passe qui sera utilisé pour le réglage Confidentiel. Ce réglage est disponible uniquement si vous sélectionnez Confidentiel, Mémoriser ou Mémoriser et imprimer lors de la sélection d'une boîte à documents privée. Si vous définissez le mot de passe d'un fichier spécifique, vous devez entrer le mot de passe pour imprimer ou effacer le fichier.

Imprimer à
Sélectionnez ce réglage pour définir l'heure de début d'impression de l'imprimante. Vous pouvez utiliser ce réglage uniquement si vous sélectionnez Programme d'impression.


Compte travaux

Compte travaux vous permet d'associer les informations d'identification d'utilisateur et de compte à chaque document imprimé.

▪ Permission utilisateur
Si un utilisateur clique sur cette option, la validité des tâches d'impression est déterminée sur base du niveau d'accès et de contrôle de l'utilisateur.

▪ Permission de groupe
Si un utilisateur clique sur cette option, la validité des tâches d'impression est déterminée sur base du niveau d'accès et de contrôle du groupe dont fait partie l'utilisateur.

Administration des paramètres des comptes travaux
Vous pouvez activer cette option et configurer les paramètres des comptes travaux afin de partager ces informations avec les clients.

Type de compte
Vous pouvez sélectionner un type de compte spécifié par l'administrateur de votre système.

Compte
Vous pouvez activer l'authentification d'impression par ID utilisateur et mot de passe.

ID uniquement
Vous pouvez activer l'authentification d'impression par ID utilisateur uniquement.


Cryptage des tâches

Si l'utilisateur sélectionne Cryptage des tâches, toutes les données PDL seront cryptées pour des raisons de sécurité.
