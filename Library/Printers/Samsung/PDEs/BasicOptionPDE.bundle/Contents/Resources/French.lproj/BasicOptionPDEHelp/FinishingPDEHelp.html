Options de tri

Options de tri vous permet de trier facilement les sorties imprimées.

▪ Aucun

▪ Décalage : En cas d'impression de plusieurs copies, la position de sortie de chaque autre jeu est décalé d'une certaine valeur.

▪ Pivoter : En cas d'impression de plusieurs copies, la position de sortie de chaque autre jeu pivote de 90 degrés.


Options d'agrafage

Options d'agrafage vous permet de spécifier le nombre d'agrafes et leur position

*Cette option est uniquement disponible si le module de finition optionnel est installé.


Options de perforation

Options de perforation vous permet de spécifier le nombre de perforations et leur position.

Les sorties imprimées sont perforées en vue de leur classement.

*Cette option est uniquement disponible si le kit de perforation optionnel est installé dans le module de finition optionnel.


Mode de sortie

▪ Mode Boîte aux lettres : Les sorties imprimées sont empilées dans le bac désigné. Si la boîte aux lettres est configurée sur Bac 3, les sorties imprimées seront empilées dans le Bac 3.

▪ Mode Séparateur de jobs : Les sorties imprimées sont empilées dans l'ordre des jobs dans chaque bac. Cependant, si un bac reste vide, celui-ci est prioritaire, ce qui signifie que les sorties imprimées seront empilées en premier lieu dans ce bac.

▪ Mode Trieuse/Interclasseuse : Les sorties imprimées sont empilées par jeux pour correspondre à la séquence des originaux dans chaque bac.

▪ Mode Empilage : Les sorties imprimées sont empilées une par une dans les bacs. Lorsque le Bac 1 est plein, les sorties imprimées sont empilées dans le Bac 2.
