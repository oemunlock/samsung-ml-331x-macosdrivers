Quality

Document Type

• Standard : Recommended for general-purpose documents with a combination of text and images.
• Photo : Recommended for photographs or images.
• Business Graphic : Recommended for office handouts such as presentation materials.
• CAD : Recommended for documents with fine lines and high detail.
• Web Page : Recommended for printing web pages.

Quality

Quality defines the quality of the print output in the simplest form.

Toner Save Mode

The Toner Save setting allows the user to save printing toner when printing a draft version of a document. Select Off to print in normal mode and select On to print with less toner applied to the page.

Darkness

The Darkness option adjusts the toner density on print outputs. The values are Normal, Light, and Dark. The default value is Normal.

Clear Text

When the Clear Text is selected, double byte characters in your document print darker than original document.

Edge Enhance

Edge Enhance allow to print the edge more smoothly on halftone(text or graphic) drawing.

Trapping

Trapping allow to reduce white-gap that may arise between one color layer and other when print color image.


Color

Color Mode

Color Mode allows the user to print an image colored or grayscale. The default value is Color.

RGB Color

• Standard : This mode is the most widely used setting (the default setting).
The color tone of this mode is the closest to that of standard monitor (sRGB Monitor).
For the best result, please change the color monitor temperature setting to 6500K and the gamma value setting to 2.2.

• Vivid : This mode is appropriate for printing presentation data and charts.
The color tone of this mode produces more vivid color images than that of Standard mode.

• Device : This mode is best used for printing fine lines, small characters and color drawings.
The color tone of this mode may be different from that of monitor.
The pure toner colors are used for printing Cyan, Magenta and Yellow colors.

• Corporate Imaging : Most common color setting used in the corporate environment.

Screen

Screen allows the user to select options to convert a continuous-tone image into printable binary image.

• Printer default : This mode will use the screen mode value in the printer control panel setting.
This mode will generate printouts with smooth tone over the entire page.

• Enhanced : This mode will generate printouts with sharp texts and smooth graphic/raster objects in the page.

• Detailed : This mode will generate printouts with sharp details over the entire page.


Levels

Using the Brightness/Contrast sliders is the easiest way to make simple adjustments to the tonal range of the image. These controls adjust all pixel values in the image at once - highlights, shadows, and midtones.
The Saturation slider lets you adjust the vividnessof an image.
The color balance sliders (Cyan-Red, Magenta-Green, Yellow-Blue) let you change the mixture of colors in an image

Paper

Paper Type specifies the type of selected paper source installed in the printer.

Skip Blank Pages
This option allows you to specify whether to skip blank pages when printing documents that consist of blank pages.

Paper Curl Reduction
This option will optimize environmental factors to reduce the paper curl level. Paper curl occurs due to many different factors such as temperature, humidity, paper type, high coverage printing and etc.

Reverse Two-Sided Printing allows you to select general print order compare to duplex print order.

Transparency Separator
• No Separator : Does not place separator sheets between transparencies.
• Printed Separator : Places the same image on the divider sheets as printed on the transparency.
• Blank Separator : Places a blank sheet between transparencies.
