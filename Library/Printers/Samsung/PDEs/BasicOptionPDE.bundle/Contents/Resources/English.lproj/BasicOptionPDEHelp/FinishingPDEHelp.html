Sort Options

Sort Options allow you to sort out the printouts easiliy.

▪ None

▪ Offset : When printing multiple copies, the output position of every other set is shifted a certain amount.

▪ Rotate : When printing multiple copies, the output positon of every other set is rotated by 90 degrees.


Staple options

Staple options allow you to specify the number of staples and the statple position

*This option is available only when the optional finisher is installed.


Hole Punch Options

Hole Punch Options allow you to specify the number of holes to punch and the punching position.

Heres are punched in ther printouts for filing them.

*This option is available only when the optional punch kit is installed in the optional finisher.


Output Mode

▪ Mailbox Mode: Printouts are stacked in the bin you have set. If you set the mailbox to Bin3, printouts will be stacked in Bin3.

▪ Job Separator Mode: Printouts are stacked in the sequence of job order in each bin. However, if there is a vacant bin left, the vacant bin has priority, which means printouts will be stacked there first.

▪ Sorter/Collator Mode: Printouts are stacked in sets to match the sequence of originals in each bin.

▪ Stacking Mode: Printouts are stacked in bins one-by-one. When Bin1 is full, printouts will be stacked in Bin2.
