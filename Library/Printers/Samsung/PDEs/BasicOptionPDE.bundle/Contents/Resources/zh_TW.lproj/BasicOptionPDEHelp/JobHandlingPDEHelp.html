列印模式

列印模式可讓您選擇所需列印方式或將列印的檔案儲存在記憶體（如大容量記憶體、印表機記憶體、內部快間記憶體、RAM 磁碟等）中。 您可以依需求選擇以下其中一種設定：


一般
選擇此設定會以一般情況列印，不會將文件儲存在印表機記憶體中。

機密
選擇此設定可讓您指定私人或機密文件的密碼並將文件儲存在 印表機記憶體中，以避免其他使用者取得這些文件。 列印文件時，您必須輸入密碼。列印後，該文件會從印表機記憶體中清除。

儲存
選取此設定可將文件直接儲存在硬碟中，而不列印文件。 當您想將文件儲存在私人的文件匣中時，您必須輸入密碼。 當您想儲存私人或機密文件時，可以使用此設定將文件儲存在私人文件匣中。

儲存和列印
選取此設定可將文件儲存在硬碟中並列印文件。 當您想將文件儲存在私人的文件匣中時，您必須輸入密碼。 當您想儲存私人或機密文件時，可以使用此設定將文件儲存在私人文件匣中。

列印排程
選擇此設定可在指定時間列印文件。 如果您將時間設定在印表機不常使用的晚上 11 點， 印表機就會將文件儲存在印表機記憶體中，待晚上 11 點再列印。

下列是可用的工作內容選項。

工作名稱
輸入列印工作的名稱。當您想稍後再列印檔案或想使用印表機的控制面板尋找儲存的檔案時，可以使用此選項。

使用者 ID
輸入使用者的 ID。 當您想稍後再列印檔案或想使用印表機的控制面板尋找儲存的檔案時，可以使用此選項。

密碼
輸入機密設定要用的密碼。 此設定只有在您選擇機密、儲存或儲存和列印與選擇儲存到任何私人文件匣時才可以使用。 如果您為某個特定檔案設定了密碼，您必須輸入密碼才能列印或刪除該檔案。

列印時間
選取此設定可設定印表機開始列印的時間。只有在您選擇了 [列印排程] 時，才可以使用此設定。


工作帳戶

工作帳戶可讓您關聯每一份列印文件的使用者和帳戶身分資訊。

▪ 使用者權限
如果使用者點選了這個選項，是否可列印工作就會依照使用者的存取和控制權限來決定。

▪ 群組權限
如果使用者點選了這個選項，是否可列印工作就會依照使用者所屬群組的存取和控制權限來決定。

管理工作帳戶設定
您可啟用此選項並設定工作帳戶設定，並與用戶端共用工作帳戶資訊。

帳戶類型
您可選取系統管理員所指定的帳戶類型。

帳戶
您可藉由使用者 ID 和密碼來啟用列印驗證機制。

僅 ID
您可藉由使用者 ID 來啟用列印驗證機制。


工作加密

如果使用者選取了工作加密，基於安全考量，所有的 PDL 資料都將加密處理。
