Modo de Impressão

O modo de Impressão permite seleccionar como imprimir ou guardar o ficheiro de impressão ao utilizar o armazenamento (Dispositivo de armazenamento em massa, Memória da impressora, Dipositivo flash interno, Disco RAM). Seleccione a opção pretendida a partir das definições seguintes:


Normal
Seleccione esta definição para uma impressão normal sem armazenar os documentos no armazenamento da impressora.

Confidencial
Seleccione esta definição para enviar o seu documento privado ou confidencial com uma palavra-passe específica para o armazenamento da impressora para que possa impedir os outros utilizadores de acederem ao mesmo. Para imprimir o documento, tem de introduzir a palavra-passe. Após imprimir, o documento é eliminado do armazenamento da impressora.

Armazenar
Seleccione esta definição para guardar um documento no disco rígido sem imprimir. Quando pretender guardar o documento numa caixa de documentos privada, tem de introduzir a palavra-passe. Utilize esta definição com uma caixa de documentos privada quando armazena um documento privado ou confidencial.

Armazenar e Imprimir
Seleccione esta definição para guardar um documento no disco rígido e imprimi-lo. Quando pretender guardar o documento numa caixa de documentos privada, tem de introduzir a palavra-passe. Utilize esta definição com uma caixa de documentos privada quando armazena um documento privado ou confidencial.

Programar Impressão
Seleccione esta definição para imprimir o documento a uma hora específica. Se definir a hora para 11:00 quando a impressora raramente é utilizada, a impressora armazena o documento no armazenamento da impressora e imprime-o às 11:00.

As opções Propriedades do Trabalho estão disponíveis da seguinte forma.

Nome do Trabalho
Introduza o nome do trabalho de impressão Pode ser utilizado quando imprime o ficheiro posteriormente ou quando encontra o ficheiro guardado utilizando o painel de controlo da impressora.

ID de Utilizador
Introduza a ID do utilizador. Pode ser utilizado quando imprime o ficheiro posteriormente ou quando encontra o ficheiro guardado utilizando o painel de controlo da impressora.

Palavra-passe
Introduza a palavra-passe que será utilizada para a definição Confidencial. Esta definição só está disponível quando selecciona Confidencial, Armazenar ou Armazenar e Imprimir ao seleccionar qualquer caixa de documentos privados. Se definir a palavra-passe num ficheiro específico, tem de introduzir a palavra-passe para imprimir ou eliminar esse ficheiro.

Imprimir Às
Seleccione esta definição para definir a hora quando a impressora começa a imprimir. Só pode utilizar esta definição se seleccionar Programar Impressão


Conta de tarefas

A Conta de tarefas permite associar as informações de identificação Utilizador e Conta com cada cada documento que imprime.

▪ Autorização de utilizador
Se um utilizador clicar nesta opção, a validade do trabalho de impressão é determinada com base no nível de acesso e controlo do utilizador.

▪ Autorização de grupo
Se um utilizador clicar nesta opção, a validade do trabalho de impressão é determinada com base no nível de acesso e controlo do grupo do qual o utilizador é um membro.

Definições da conta de tarefas Admin
Pode activar esta opção e configurar as definições da conta de tarefas para partilhar informações da conta de tarefas com clientes.

Tipo de conta
Pode seleccionar um tipo de conta especificado pelo seu administrador de sistema.

Conta
Pode activar a autenticação de impressão através da ID de utilizador e palavra-passe.

Apenas ID
Só pode activar a autenticação de impressão através da ID de utilizador.


Encriptação de Trabalhos

Se o utilizador seleccionar Encriptação de Trabalhos, todos os dados PDL serão encriptados para segurança.
