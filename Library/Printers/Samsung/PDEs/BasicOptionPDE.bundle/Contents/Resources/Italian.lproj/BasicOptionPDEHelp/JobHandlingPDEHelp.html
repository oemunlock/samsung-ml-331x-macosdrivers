Modalità di stampa

Modalità di stampa consente di selezionare come stampare o salvare il file di stampa utilizzando i supporti di memorizzazione (dispositivo di memorizzazione di massa, memoria della stampante, unità flash interna o disco RAM). Selezionare una delle impostazioni seguenti:


Normale
Selezionare questa impostazione per la stampa normale senza memorizzare i documenti nella memoria della stampante.

Riservato
Selezionare queste impostazione per inviare il documento privato o riservato con la password specificata alla memoria della stampante, in modo tale da poter impedire l'accesso agli altri utenti. Per stampare il documento è necessario immettere la password. Dopo aver stampato, il documento viene cancellato dalla memoria della stampante.

Memorizza
selezionare questa impostazione per salvare un documento nel disco rigido senza stampare. Se si desidera salvare il documento in una casella documenti privata, è necessario immettere la password. Usare questa impostazione con una casella documenti privata per memorizzare un documento privato o riservato.

Memorizza e stampa
selezionare questa impostazione per salvare un documento nel disco rigido e stampare. Se si desidera salvare il documento in una casella documenti privata, è necessario immettere la password. Usare questa impostazione con una casella documenti privata per memorizzare un documento privato o riservato.

Pianificazione stampa
selezionare questa impostazione per stampare il documento in un determinato momento. Se si imposta l'ora su 23:00 quando la stampante viene usata poco, essa memorizza il documento nella memoria della stampante e stampa alle ore 23:00.

Sono disponibili le seguenti opzioni di proprietà del lavoro.

Nome lavoro
immettere il nome del lavoro di stampa. È possibile utilizzarlo quando si stampa il file in un secondo momento o per trovare il file salvato utilizzando il pannello di controllo della stampante.

ID utente
immettere l'ID dell'utente. È possibile utilizzarlo quando si stampa il file in un secondo momento o per trovare il file salvato utilizzando il pannello di controllo della stampante.

Password
Immettere la password che verrà utilizzata per l'impostazione Riservato. Questa impostazione è disponibile solo quando si seleziona Riservato, Memorizza o Memorizza e stampa selezionando una qualsiasi casella documenti privata. Se viene impostata la password per un file specifico, sarà necessario immetterla per stampare o eliminare il file.

Stampa alle
Selezionare questa impostazione per impostare l’ora in cui iniziare la stampa. Questa impostazione può essere utilizzata solo se si seleziona Pianificazione stampa.


Account lavori

Account lavori consente di associare informazioni che identificano utente e account a ogni documento stampato.

▪ Autorizzazione utente
Se un utente fa clic su questa opzione, la validità del lavoro di stampa viene determinata in base al livello dell’accesso e del controllo dell’utente.

▪ Autorizzazione del gruppo
Se un utente fa clic su questa opzione, la validità del lavoro di stampa viene determinata in base al livello dell’accesso del gruppo e del controllo di cui fa parte l’utente.

Impostazioni account lavori Admin
È possibile selezionare questa opzione e specificare le impostazioni dell'account dei lavori per condividere le informazioni amministrative con i clienti.

Tipo di account
È possibile selezionare un tipo di account specificato dal proprio amministratore del sistema.

Account
È possibile attivare l'autenticazione di stampa tramite l'ID utente e la password.

Solo ID
È possibile attivare l'autenticazione di stampa tramite solo l'ID utente.


Crittografia lavoro

Se l'utente seleziona Crittografia lavoro, tutti i dati PDL verranno crittografati per garantire la sicurezza.
