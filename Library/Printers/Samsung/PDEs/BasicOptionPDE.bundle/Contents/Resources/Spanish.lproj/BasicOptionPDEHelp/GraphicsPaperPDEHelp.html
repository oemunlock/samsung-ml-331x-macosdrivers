Tipo de documento

• Estándar: recomendado para documentos generales, con una combinación de texto e imágenes.
• Fotografía: recomendado para fotografías e imágenes.
• Gráfica empresarial: recomendado para material impreso de oficina, como por ejemplo materiales para presentaciones.
• CAD: recomendado para documentos con líneas finas y gran nivel de detalle.
• Página web: recomendado para imprimir páginas web.

Calidad

Calidad define la calidad de la copia impresa de la forma más sencilla posible.

Modo Ahorro de tóner

El ajuste Ahorro de tóner permite al usuario ahorrar tóner de impresión al imprimir una versión borrador de un documento. Seleccione Desactivado para imprimir en modo normal y seleccione Activado para imprimir aplicando menos tóner a la página.

Oscuridad

La opción Oscuridad ajusta la densidad del tóner en las impresiones. Los posibles valores son Normal, Claro y Oscuro. El valor predeterminado es Normal.

Texto sin cifrar

Cuando se seleccione Texto sin cifrar los caracteres de doble byte del documento se imprimirán en un tono más oscuro que el del documento original.

Mejora de bordes

Mejora de bordes permite imprimir el borde con mayor suavidad en dibujos de medios tonos (texto o gráfico).

Reventado

Reventado permite reducir los espacios en blanco que podrían surgir entre una capa de color y otra a la hora de imprimir imágenes a color.


Color

Modo color

Modo color permite al usuario imprimir una imagen a color o en escala de grises. El valor predeterminado es Color.

Color RGB

• Estándar: Este modo es el ajuste más utilizado (es el ajuste predeterminado).
El tono de color que se logra en este modo es el más próximo posible al de un monitor convencional (monitor sRGB).
Para lograr los mejores resultados posibles, cambie el ajuste de temperatura de color del monitor a 6500K, y el ajuste de valor de gamma a 2.2.

• Vivo: este modo resulta adecuado para imprimir datos de presentaciones y gráficos.
El tono de color de este modo genera imágenes de colores más vivos que las del modo Estándar.

• Dispositivo: este modo es el más utilizado para imprimir líneas finas, caracteres pequeños y dibujos en color.
El tono de color de este modo podría ser diferente al del monitor.
Los colores de tóner puros que se utilizan para imprimir son el cian, el magenta y el amarillo.

• Imagen corporativa: este ajuste de color es el más común para los entornos corporativos.

Pantalla

Pantalla permite al usuario seleccionar diversas opciones para convertir una imagen de tono continuo a una imagen binaria imprimible.

• Impresora predeterminada: este modo utilizará el valor del modo en pantalla en el ajuste del panel de mandos de la impresora.
Este modo genera impresiones con un tono uniforme en toda la página.

• Mejorado: este modo genera impresiones con textos nítidos y objetos de fotos/ráster uniformes en la página.

• Detallado: este modo genera impresiones con detalles nítidos en toda la página.


Niveles

El uso de los deslizadores Brillo/Contraste es la forma más fácil de realizar ajustes sencillos en el rango tonal de la imagen. Estos controles ajustan todos los valores de píxel de la imagen (destacados, sombras y tonos medios) de forma simultánea.
El deslizador Saturación le permite ajustar la viveza de una imagen.
Los deslizadores de balance de color (cian-rojo, magenta-verde, amarillo-azul) le permiten cambiar la mezcla de colores de una imagen.

Papel

Tipo de papel especifica el tipo de origen de papel seleccionado instalado en la impresora.

Saltar páginas en blanco
Esta opción le permite especificar si desea omitir las páginas en blanco al imprimir documentos que incluyen páginas en blanco.

Reduc. papel abarq.
Esta opción optimizará factores medioambientales para reducir el nivel de abarquillamiento de papel. El abarquillamiento de papel aparece debido a diversos factores diferentes, como la temperatura, la humedad, el tipo de papel, la impresión de alta cobertura, etc.

Impresión a dos caras inversa le permite seleccionar el orden de impresión general, en comparación con el orden de impresión de doble cara.

Separador de transparencias
• Sin separador: no coloca separadores entre las transparencias.
• Separador impreso: coloca la misma imagen en las hojas divisoras, tal y como se imprimen en la transparencia.
• Separador blanco: coloca una hoja en blanco entre las transparencias.
