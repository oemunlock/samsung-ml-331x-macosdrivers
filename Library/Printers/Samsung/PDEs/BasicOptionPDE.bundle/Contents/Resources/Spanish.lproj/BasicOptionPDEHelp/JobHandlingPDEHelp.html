Modo de impresión

Modo de impresión le permite seleccionar cómo desea imprimir o guardar el archivo imprimido utilizando el almacenamiento (dispositivo de almacenamiento masivo, memoria de la impresora, unidad flash interna, disco RAM). Seleccione uno de los ajustes siguientes:


Normal
Seleccione este ajuste para impresión normal sin almacenar sus documentos en el almacenamiento de la impresora.

Confidencial
Seleccione este ajuste para enviar su documento privado o confidencial, con una contraseña especificada, al almacenamiento de la impresora, para impedir que otros usuarios accedan a él. Para imprimir el documento deberá introducir la contraseña. Una vez completado el proceso de impresión, el documento se eliminará del almacenamiento de la impresora.

Guardar
Seleccione este ajuste para guardar un documento en el disco duro sin imprimirlo. Si desea guardar su documento en un buzón de documentos privado, deberá introducir la contraseña. Utilice este ajuste con un buzón de documentos privado siempre que guarde un documento privado o confidencial.

Guardar e imprimir
Seleccione este ajuste para guardar un documento en el disco duro e imprimirlo. Si desea guardar su documento en un buzón de documentos privado, deberá introducir la contraseña. Utilice este ajuste con un buzón de documentos privado siempre que guarde un documento privado o confidencial.

Programa de impresión
Seleccione este ajuste para imprimir el documento en un momento concreto. Si la impresora está muy solicitada y establece la hora de impresión en las 11 PM, la impresora guardará el documento en el almacenamiento de la impresora y lo imprimirá a las 11 PM.

Las opciones de Propiedades del trabajo son las siguientes.

Nombre del trabajo
Introduzca el nombre del trabajo de impresión. Podrá utilizar este nombre cuando imprima el archivo posteriormente, o para buscar el archivo guardado con el panel de mandos de la impresora.

ID usuario
Introduzca el identificador del usuario. Podrá utilizar este nombre cuando imprima el archivo posteriormente, o para buscar el archivo guardado con el panel de mandos de la impresora.

Contraseña
Introduzca la contraseña que se utilizará para el ajuste Confidencial. Este ajuste está disponible únicamente si selecciona Confidencial, Guardar o Guardar e imprimir seleccionando un buzón de documentos privado. Si establece la contraseña en un archivo concreto, deberá introducir esta contraseña para poder imprimir o eliminar el archivo.

Imprimir a las
Seleccione este ajuste para establecer la hora a la que debe comenzar el trabajo de impresión. Este ajuste solo podrá utilizarlo si selecciona Programación de impresión.


Cuenta de trabajos

Cuenta de trabajos le permite asociar la información de identificación de cuenta y de usuario a cada documento que imprime.

▪ Permiso de usuario
Si un usuario hace clic en esta opción, la validez del trabajo de impresión se determinará en función del nivel de acceso y control del usuario.

▪ Permisos de grupo
Si un usuario hace clic en esta opción, la validez del trabajo de impresión se determinará en función del nivel de acceso y control del grupo del que el usuario forma parte.

Configuración de cuenta de trabajos de administración
Puede habilitar esta opción y configurar los ajustes de cuenta de trabajos para compartir la información de cuenta de trabajos con sus clientes.

Tipo de cuenta
Puede seleccionar uno de los tipos de cuenta especificados por el administrador del sistema.

Cuenta
Puede habilitar la autenticación de la impresión por identificación de usuario y contraseña.

Solo ID
Puede habilitar la autenticación de la impresión solo por identificación de usuario.


Cifrado de trabajos

Si el usuario selecciona Cifrado de trabajos se cifrarán todos los datos PDL por razones de seguridad.
