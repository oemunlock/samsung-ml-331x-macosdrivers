Druckmodus

Druckmodus ermöglicht Ihnen auszuwählen, wie die Druckdatei zu drucken oder zu speichern ist (Massenspeichergerät, Druckerspeicher, Internes Flash-Laufwerk, RAM-Disk). Wählen Sie unter den folgenden Möglichkeiten die gewünschte Einstellung aus:


Normal
Wählen Sie diese Einstellung, wenn Sie normal drucken wollen, ohne dass Ihre Dokumente im Druckerspeicher gespeichert werden.

Vertraulich
Wählen Sie diese Einstellung, wenn Sie Ihre privaten oder vertraulichen Dokumente kennwortgeschützt im Druckerspeicher ablegen wollen, so dass andere Benutzer kein Zugriff darauf haben. Zum Drucken des Dokuments müssen Sie dann das Kennwort eingeben. Nach dem Drucken wird das Dokument aus dem Druckerspeicher gelöscht.

Speichern
Wählen Sie diese Einstellung, um das Dokument auf der Festplatte zu speichern, ohne es zu drucken. Wenn Sie Ihr Dokument in einem privaten Dokumentfach speichern wollen, müssen Sie das Kennwort eingeben. Verwenden Sie diese Einstellung in Verbindung mit einem privaten Dokumentfach, wenn Sie ein privates oder vertrauliches Dokument speichern.

Speichern und drucken
Wählen Sie diese Einstellung, um ein Dokument auf der Festplatte zu speichern und zu drucken. Wenn Sie Ihr Dokument in einem privaten Dokumentfach speichern wollen, müssen Sie das Kennwort eingeben. Verwenden Sie diese Einstellung in Verbindung mit einem privaten Dokumentfach, wenn Sie ein privates oder vertrauliches Dokument speichern.

Druckzeitplan
Wählen Sie diese Einstellung, um das Dokument zu einem bestimmten Zeitpunkt zu drucken. Wenn Sie die Zeit zum Beispiel auf 23 Uhr abends festlegen, wenn der Drucker kaum benutzt wird, speichert der Drucker das Dokument im Druckerspeicher und druckt es zur festgelegten Zeit.

Folgende Optionen stehen für die Auftragseigenschaften zur Verfügung.

Name des Druckauftrags
Geben Sie Namen des Druckauftrags ein. Diese Option kann verwendet werden, wenn Sie eine Datei zu einem späteren Zeitpunkt drucken oder die gespeicherte Datei über das Bedienfeld des Druckers angeben wollen.

Benutzer-ID
Geben Sie die Benutzer-ID ein. Diese Option kann verwendet werden, wenn Sie eine Datei zu einem späteren Zeitpunkt drucken oder die gespeicherte Datei über das Bedienfeld des Druckers angeben wollen.

Kennwort
Geben Sie das Kennwort ein, das für die Einstellung Vertraulich festgelegt ist. Steht nur zur Verfügung, wenn Vertraulich, Speichern oder Speichern und Drucken mit Auswahl eines persönlichen Dokumentenfachs ausgewählt ist. Wenn Sie ein Kennwort für eine bestimmte Datei festlegen, müssen Sie dieses eingeben, wenn Sie diese Datei drucken oder löschen wollen.

Drucken um
Wählen Sie diese Einstellung, um die Zeit festzulegen, zu welcher der Drucker den Druck beginnen soll. Diese Einstellung steht nur bei Auswahl der Option Druckzeitplan zur Verfügung.


Auftragsprotokollierung

Auftragsprotokollierung ermöglicht Ihnen, jedes gedruckte Dokument einer Benutzer-ID und entsprechenden Benutzerkonten und deren Daten zuzuordnen.

▪ Benutzerberechtigung
Wenn Sie diese Option aktivieren, wird jeder Druckauftrag daraufhin geprüft, ob der betreffende Benutzer auf Basis der ihm zugeordneten Benutzer- und Zugriffsrechte entsprechende Befugnis hat.

▪ Gruppenberechtigung
Wenn Sie diese Option aktivieren, wird jeder Druckauftrag daraufhin geprüft, ob der betreffende Benutzer auf Basis der Benutzer- und Zugriffsrechte der Gruppe, bei der er Mitglied ist, entsprechende Befugnis hat.

Administrator-Auftragsprotokollierungseinstellungen
Wenn Sie diese Option aktivieren, können Sie Einstellungen für die Auftragsprotokollierung konfigurieren, um die Druckauftragsdaten mit Kunden zu teilen.

Auftragsprotokollierungstyp
Sie können einen Auftragsprotokollierungstyp auswählen, der von Ihrem Systemadministrator spezifiziert ist.

Benutzerkontenführung
Sie können festlegen, dass zum Drucken eine Benutzer-ID und ein Kennwort eingegeben werden müssen.

Nur ID
Sie können festlegen, dass zum Drucken eine Benutzer-ID, aber kein Kennwort eingegeben werden muss.


Auftragsverschlüsselung

Wenn Sie Auftragsverschlüsselung auswählen, werden alle PDL-Daten zu Sicherheitszwecken verschlüsselt.
