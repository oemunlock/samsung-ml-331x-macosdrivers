[As opções de definições podem variar consoante as impressoras e a versão do SO do Mac.]

Caixa de diálogo Definição do trabalho

Modo de impressão

Modo de impressão permite seleccionar como imprimir ou guardar o ficheiro de impressão usando o armazenamento (disco rígido opcional ou memória da impressora). Seleccione a definição pretendida das seguintes opções:


Normal

Seleccione esta definição para uma impressão normal sem armazenar os seus documentos no armazenamento da impressora.

Confidencial

Seleccione esta definição para enviar o seu documento privado ou confidencial com uma palavra-passe especificada para o disco rígido, para que possa evitar que outros utilizadores lhe tenham acesso. Para imprimir o documento, tem de introduzir a palavra-passe. Depois de impresso, o documento é limpo do disco rígido. Se pretender guardar o ficheiro no disco rígido depois da impressão, assinale a caixa de verificação Armazenar em HDD.

Armazenar

Seleccione esta definição para guardar um documento no disco rígido sem o imprimir.

Spool

Esta opção pode ser útil para lidar com uma grande quantidade de dados. Se seleccionar esta definição, a impressora envia o documento para o spool do disco rígido e depois imprime-o a partir da fila de espera do disco, reduzindo a carga no computador.

Armazenar e imprimir

Seleccione esta definição para guardar um documento no disco rígido e imprimi-lo.

Programa de impressão

Seleccione esta definição para imprimir o documento a uma data e hora especificada. Se definir a hora para as 23:00 quando a impressora está a ser pouco utilizada, a impressora guarda o documento no disco rígido e imprime-o às 23:00.



As opções de Propriedades do trabalho estão disponíveis como se segue.

ID de Utilizador

Introduza o ID do utilizador. Pode ser utilizado quando imprime o ficheiro mais tarde ou quando encontra o ficheiro guardado utilizando o painel de controlo da impressora.

Nome do trabalho

Introduza o nome do trabalho de impressão. Pode ser utilizado quando imprime o ficheiro mais tarde ou quando encontra o ficheiro guardado utilizando o painel de controlo da impressora.

Palavra-passe

Introduza a palavra-passe que será utilizada para a definição Confidencial.

Imprimir em

Seleccione esta opção para definir a hora quando a impressora começa a impressão. Apenas pode utilizar esta definição se seleccionou Diferimento.
