[Le opzioni di impostazione possono variare a seconda delle stampanti e della versione di Mac OS.]

Finestra di dialogo comando Impostazione lavoro

Modalità di stampa

Modalità di stampa consente di selezionare come stampare o salvare il file di stampa utilizzando la memoria (disco rigido opzionale o memoria della stampante). Selezionare una delle impostazioni seguenti:


Normale

Selezionare questa impostazione per la stampa normale senza memorizzare i documenti nella memoria della stampante.

Riservato

Selezionare questa impostazione per inviare il documento privato o riservato con la password specificata alla memoria della stampante, in modo tale da poter impedire l'accesso agli altri utenti. Per stampare il documento è necessario immettere la password. Dopo aver stampato, il documento viene cancellato dalla memoria della stampante.

Memorizza

Selezionare questa impostazione per salvare un documento nel disco rigido senza stamparlo.

Spooling

Questa opzione può essere utile per gestire una grande quantità di dati. Se si seleziona questa impostazione, viene eseguito lo spooling del documento sul disco rigido, quindi esso viene stampato dalla coda del disco rigido, diminuendo il carico di lavoro del computer.

Memorizza e stampa

Selezionare questa impostazione per salvare un documento nel disco rigido senza stamparlo.

Pianificazione stampa

Selezionare questa impostazione per stampare il documento all’ora specificata. Se si imposta l'ora su 23:00 quando la stampante viene usata poco, essa memorizza il documento nella memoria della stampante e stampa alle ore 23:00.



Sono disponibili le seguenti opzioni di proprietà del lavoro.

ID utente

Immettere l'ID dell'utente. Può essere usata per stampare il file in un secondo momento oppure quando si trova il file salvato utilizzando il pannello di controllo della stampante.

Nome lavoro

Immettere il nome del lavoro di stampa. Consente di stampare il file in un secondo momento oppure quando si trova il file salvato utilizzando il pannello di controllo della stampante.

Password

Immettere la password che verrà utilizzata per l'impostazione Riservato.

Stampa alle

Selezionare questa impostazione per impostare l'ora in cui la stampante inizia a stampare. Questa impostazione può essere utilizzata solo se si seleziona Ritardo.
