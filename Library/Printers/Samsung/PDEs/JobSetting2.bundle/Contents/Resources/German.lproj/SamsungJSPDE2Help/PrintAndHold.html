[Je nach Drucker und Mac OS-Version unterscheiden sich eventuell die angebotenen Einstellungen.]

Auftragseinstellung

Druckmodus

Unter „Druckmodus“ wählen Sie aus, wie eine Datei über das Speichermedium (optionale Festplatte oder Druckerspeicher) gedruckt oder gespeichert werden soll. Wählen Sie unter den folgenden die gewünschte Einstellung aus:


Normal

Wählen Sie diese Einstellung, wenn die Datei auf herkömmliche Art gedruckt werden soll, d. h. die Dokumente werden nicht im Speichermedium des Druckers aufbewahrt.

Vertraulich

Wählen Sie diese Einstellung, wenn ein privates oder vertrauliches Dokument passwortgeschützt im Speichermedium des Druckers aufbewahrt werden soll, sodass andere Benutzer nicht darauf zugreifen können.

Speichern

Wählen Sie diese Einstellung, um das Dokument auf der Festplatte zu speichern, ohne es zu drucken.

Spool

Diese Option kann bei großen Datenmengen nützlich sein. Wenn Sie diese Einstellung auswählen, wird das Dokument während des Druckens auf der Festplatte zwischengespeichert und über die Festplattenwarteschlange gedruckt. Auf diese Weise wird die Auslastung des Computers verringert.

Speichern und drucken

Wählen Sie diese Einstellung, um ein Dokument auf der Festplatte zu speichern und zu drucken.

Druckplan

Wählen Sie diese Einstellung, wenn Sie das Dokument zu einer bestimmten Zeit drucken möchten. Wenn Sie die Uhrzeit auf 23 Uhr einstellen, wenn der Drucker kaum genutzt wird, wird das Dokument im Speichermedium des Druckers aufbewahrt und um 23 Uhr gedruckt.



Folgende Optionen können für die Auftragseigenschaften eingerichtet werden:

Benutzer-ID

Geben Sie die Kennung des Benutzers ein. Diese Option eignet sich, um die Datei zu einem späteren Zeitpunkt oder, indem Sie die Datei über das Bedienfeld auswählen, auszudrucken.

Auftragsname

Geben Sie den Namen des Druckauftrags ein. Diese Option eignet sich, um die Datei zu einem späteren Zeitpunkt oder, indem Sie die Datei über das Bedienfeld auswählen, auszudrucken.

Passwort

Geben Sie hier das Passwort für Dokumente mit der Einstellung Vertraulich ein.

Drucken um

Wählen Sie diese Einstellung, um die Druckzeit festzulegen. Diese Einstellung kann erst nach Aktivieren der Option „Verzögerung“ ausgewählt werden.
