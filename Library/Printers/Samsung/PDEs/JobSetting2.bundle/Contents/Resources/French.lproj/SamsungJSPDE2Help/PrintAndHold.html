[Les options de configuration peuvent varier en fonction des imprimantes et de la version de Mac OS utilisée.]

Boîte de dialogue Réglage de tâche

Mode d’impression

La boîte de dialogue Mode d’impression vous permet de choisir le mode d’impression ou d’enregistrement du fichier d’impression via la mémoire de stockage (lecteur de disque dur optionnel ou mémoire de l’imprimante). Vous pouvez sélectionner l’un des paramètres suivants :


Normal

Sélectionnez cette option pour une impression normale sans enregistrer vos documents dans la mémoire de stockage.

Confidentiel

Sélectionnez cette option paramètre pour envoyer votre document privé ou confidentiel avec un mot de passe spécifié vers la mémoire de stockage pour empêcher les autres utilisateurs d’y accéder. Pour imprimer le document, saisissez le mot de passe. Après l’impression, le document est supprimé de la mémoire de stockage.

Enregistrer

Sélectionnez cette option pour enregistrer un document sur le disque dur sans l’imprimer.

Désynchronisation

Cette option s'avère utile pour le traitement de fichiers volumineux. Si vous sélectionnez cette option, l'imprimante mémorise le document dans une file d'attente sur le disque dur, puis l'imprime depuis cette file d'attente, réduisant ainsi la charge de travail de l'ordinateur.

Stocker et imprimer

Sélectionnez cette option pour enregistrer un document sur le disque dur et l’imprimer.

Planificat impression

Sélectionnez ce paramètre pour imprimer le document à une heure spécifique. Si vous réglez l’heure sur 23 h 00, lorsque l’imprimante n’est presque pas utilisée, elle enregistre le document dans la mémoire de stockage et l’imprime à 23 h 00.



Les options de propriétés de travail sont les suivantes :

Utilisateur

Saisissez l'ID de l'utilisateur. Cette option sert à imprimer le fichier à une heure plus tardive ou à chercher un fichier enregistré à l'aide du panneau de commande de l'imprimante.

Nom travail

Saisissez le nom du travail d'impression. Cette option sert à imprimer le fichier à une heure plus tardive ou à chercher un fichier enregistré à l'aide du panneau de commande de l'imprimante.

Mot de passe

Saisissez le mot de passe qui sera utilisé dans l'option Confidentiel.

Imprimer à

Cette option vous permet de régler l'heure à laquelle l'imprimante doit commencer à imprimer. Cette option est uniquement disponible si vous sélectionnez Différer.
