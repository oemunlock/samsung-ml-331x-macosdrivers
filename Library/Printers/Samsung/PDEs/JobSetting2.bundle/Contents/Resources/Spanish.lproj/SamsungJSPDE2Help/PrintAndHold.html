Cuadro Configuración del trabajo

Modo de impresión

El Modo de impresión permite seleccionar cómo imprimir o guardar el archivo de impresión utilizando el espacio de almacenamiento (disco duro opcional o memoria de la impresora). Seleccione la configuración que desee entre las siguientes:


Normal

Seleccione este ajuste para imprimir normalmente, sin guardar los documentos en el espacio de almacenamiento de la impresora.

Confidencial

Seleccione este ajuste para enviar sus documentos confidenciales o privados al espacio de almacenamiento de la impresora con una contraseña concreta, a fin de evitar que otros usuarios puedan acceder a ellos. Para imprimir el documento, deberá introducir la contraseña. Después de imprimirlo, el documento se elimina del espacio de almacenamiento de la impresora.

Guardar

Seleccione este ajuste para guardar un documento en el disco duro sin imprimirlo.

Spool

Esta opción puede resultar útil para gestionar grandes cantidades de datos. Si selecciona este ajuste, el documento se envía a la cola del disco duro y se imprime desde allí, reduciendo así la carga de trabajo del ordenador.

Guardar e imprimir

Seleccione este ajuste para guardar un documento en el disco duro e imprimirlo.

Program. de impresión

Seleccione esta configuración para imprimir el documento a una hora determinada. Si establece la hora a las 11 PM, cuando la impresora apenas se utiliza, la impresora guarda el documento en el disco duro y lo imprimirá a las 11 PM.



Las opciones de Propiedades de la tarea disponibles son las siguientes.

Id. de usuario

Introduzca el identificador del usuario. Puede utilizarse cuando se imprime el archivo a una hora posterior o para encontrar el archivo guardado utilizando el panel de control de la impresora.

Nombre de tarea

Introduzca el nombre de la tarea de impresión. Puede utilizarse cuando se imprime el archivo a una hora posterior o para encontrar el archivo guardado utilizando el panel de control de la impresora.

Contraseña

Introduzca la contraseña que se utilizará para el ajuste Confidencial.

Imprimir a las

Seleccione este ajuste para especificar la hora a la que se iniciará la impresión. Este ajuste sólo puede utilizarse si se selecciona Retraso.
