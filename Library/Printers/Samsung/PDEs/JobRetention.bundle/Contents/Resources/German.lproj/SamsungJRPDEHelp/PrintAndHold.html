[Je nach Drucker und Mac OS-Version unterscheiden sich eventuell die angebotenen Einstellungen.]

Auftragseinstellung

Im Dialogfeld Auftragseinstellung können Sie die Druckauftragseinstellungen wählen und festlegen, ob die Druckdatei auf einer optionalen Festplatte gespeichert werden soll. Wählen Sie eine der folgenden Einstellungen:


Normal

Wählen Sie diese Einstellung, wenn die Datei auf herkömmliche Art gedruckt werden soll, d. h. die Dokumente werden nicht auf der Festplatte gespeichert. Aktivieren Sie das Kontrollkästchen Auf HDD speichern, um das Dokument nach dem Drucken auf der Festplatte zu speichern.Sie können das gespeicherte Dokument jederzeit über das Bedienfeld des Druckers oder den PC erneut ausdrucken.

Vertraulich

Wählen Sie diese Einstellung, um alle privaten und vertraulichen Dokumente, die auf der Festplatte gespeichert werden, mit einem bestimmten Passwort zu versehen, damit andere Benutzer nicht darauf zugreifen können. In der Folge müssen Sie dann das Passwort eingeben, um das Dokument zu drucken. Nach dem Drucken wird das Dokument von der Festplatte gelöscht. Wenn Sie die Datei nach dem Drucken auf der Festplatte speichern möchten, aktivieren Sie das Kontrollkästchen Auf HDD speichern.

Speichern

Wählen Sie diese Einstellung, um das Dokument auf der Festplatte zu speichern ohne es zu drucken. Wenn Sie das gespeicherte Dokument über das Bedienfeld des Druckers ausdrucken möchten, müssen Sie ein Passwort eingeben. Wählen Sie diese Einstellung, wenn Sie ein privates oder vertrauliches Dokument speichern möchten. Aktivieren Sie Auf öffentlicher HDD speichern, um ein Dokument auf der Festplatte zu speichern, ohne es zu drucken. So gespeicherte Dokumente können von alle Benutzer ohne Eingabe eines Passworts gedruckt werden. Verwenden Sie diese Einstellung, um öffentlich zugängliche Dokumente zu speichern.

Spool

Diese Option kann bei großen Datenmengen nützlich sein. Wenn Sie diese Einstellung auswählen, wird das Dokument während des Druckens auf der Festplatte zwischengespeichert und über die Festplattenwarteschlange gedruckt. Auf diese Weise wird die Auslastung des Computers verringert.

Druckplan

Wählen Sie diese Einstellung, wenn Sie das Dokument zu einer bestimmten Zeit drucken möchten. Wenn Sie z. B. als Uhrzeit 23 Uhr festlegen, wenn der Drucker wenig benutzt wird, wird das Dokument auf der Druckerfestplatte gespeichert und um 23 Uhr ausgedruckt.



Folgende Optionen können für die Auftragseigenschaften eingerichtet werden:

Benutzer-ID

Geben Sie die Kennung des Benutzers ein. Diese Option eignet sich, um die Datei zu einem späteren Zeitpunkt oder, indem Sie die Datei über das Bedienfeld auswählen, auszudrucken.

Auftragsname

Geben Sie den Namen des Druckauftrags ein. Diese Option eignet sich, um die Datei zu einem späteren Zeitpunkt oder, indem Sie die Datei über das Bedienfeld auswählen, auszudrucken.

Passwort

Geben Sie hier das Passwort für Dokumente mit der Einstellung Vertraulich ein. Diese Einstellung ist nur verfügbar, wenn Sie die Optionen Vertraulich oder Speichern, jedoch nicht Auf öffentlicher HDD speichern aktivieren. Wenn Sie für eine bestimmte Datei ein Passwort festgelegt haben, können Sie die Datei nur nach Eingabe des Passworts drucken oder löschen.

Drucken um

Wählen Sie diese Einstellung, um die Druckzeit festzulegen. Diese Einstellung kann erst nach Aktivieren der Option „Verzögerung“ ausgewählt werden.
