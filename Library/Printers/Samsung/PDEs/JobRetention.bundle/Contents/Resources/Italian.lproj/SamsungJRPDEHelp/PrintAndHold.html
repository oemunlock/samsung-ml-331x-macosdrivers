[Le opzioni di impostazione possono variare a seconda delle stampanti e della versione di Mac OS.]

Finestra di dialogo comando Impostazione lavoro

La finestra di dialogo Impostazione lavoro consente di selezionare la modalità di stampa o di salvare il file di stampa utilizzando il disco rigido opzionale.


Normale

Selezionare questa impostazione per la stampa normale senza memorizzare i documenti nel disco rigido. Selezionare la casella di controllo Memorizza nel disco rigido per salvare il documento nel disco rigido dopo la stampa. Il documento salvato può essere stampato in qualsiasi momento sia dal pannello di controllo che dal PC.

Riservato

Selezionare questa impostazione per inviare al disco rigido un documento privato o riservato con una password in modo da impedirne l'accesso ad altri utenti. Per stampare il documento è necessario immettere la password. Dopo la stampa, il documento viene cancellato dal disco rigido. Se si desidera salvare il file sul disco rigido dopo la stampa, selezionare la casella di controllo Memorizza nel disco rigido.

Memorizza

Selezionare questa impostazione per salvare un documento nel disco rigido senza stamparlo. Quando si stampa il documento salvato utilizzando il pannello di controllo è necessario immettere la password. Utilizzare questa impostazione quando si memorizza un documento privato o riservato. Selezionare Memorizza nel disco rigido pubblico per salvare un documento sul disco rigido senza stamparlo. Il documento salvato utilizzando questa impostazione può essere stampato da qualsiasi utente senza immettere la password. Usare questa impostazione quando si memorizzano documenti pubblici.

Spooling

Questa opzione può essere utile per gestire una grande quantità di dati. Se si seleziona questa impostazione, viene eseguito lo spooling del documento sul disco rigido, quindi esso viene stampato dalla coda del disco rigido, diminuendo il carico di lavoro del computer.

Pianificazione stampa

Selezionare questa impostazione per stampare il documento all'ora specificata. Se si imposta l'ora su 11 PM quando difficilmente la stampante verrà usata, la stampante memorizza il documento sul disco rigido e lo stampa alle 11 PM.



Sono disponibili le seguenti opzioni di proprietà del lavoro.

ID utente

Immettere l'ID dell'utente. Può essere usata per stampare il file in un secondo momento oppure quando si trova il file salvato utilizzando il pannello di controllo della stampante.

Nome lavoro

Immettere il nome del lavoro di stampa. Consente di stampare il file in un secondo momento oppure quando si trova il file salvato utilizzando il pannello di controllo della stampante.

Password

Immettere la password che verrà utilizzata per l'impostazione Riservato. Questa impostazione è disponibile solo quando si seleziona Riservato o Memorizza, senza selezionare Memorizza nel disco rigido pubblico. Se si imposta la password per un file specifico è necessario immetterla per stampare o eliminare il file.

Stampa alle

Selezionare questa impostazione per impostare l'ora in cui la stampante inizia a stampare. Questa impostazione può essere utilizzata solo se si seleziona Ritardo.
