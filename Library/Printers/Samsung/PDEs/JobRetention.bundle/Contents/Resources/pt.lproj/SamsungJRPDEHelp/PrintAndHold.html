[As opções de definições podem variar consoante as impressoras e a versão do SO do Mac.]

Caixa de diálogo Definição do trabalho

A caixa de diálogo Definição do trabalho permite seleccionar como imprimir ou guardar o ficheiro de impressão usando a unidade de disco rígido opcional (HDD). Seleccione a definição pretendida das seguintes opções:


Normal

Seleccione esta definição para impressão normal sem armazenar os seus documentos no disco rígido. Assinale a caixa de verificação Armazenar em HDD para guardar o documento no disco rígido depois da impressão. O documento guardado pode ser impresso em qualquer altura que quiser a partir do painel de controlo da impressora assim como no seu PC.

Confidencial

Seleccione esta definição para enviar o seu documento privado ou confidencial com uma palavra-passe especificada para o disco rígido, para que possa evitar que outros utilizadores lhe tenham acesso. Para imprimir o documento, tem de introduzir a palavra-passe. Depois de impresso, o documento é limpo do disco rígido. Se pretender guardar o ficheiro no disco rígido depois da impressão, assinale a caixa de verificação Armazenar em HDD.

Armazenar

Seleccione esta definição para guardar um documento no disco rígido sem o imprimir. Quando imprimir o documento guardado usando o painel de controlo, tem de introduzir a palavra-passe. Utilize esta definição quando armazena um documento privado ou confidencial. Assinale Armazenar em HDD público para guardar o documento no disco rígido sem o imprimir. O documento guardado utilizando esta definição pode ser impresso por qualquer utilizador sem introduzir a palavra-passe. Utilize esta definição quando armazena documentos públicos.

Spool

Esta opção pode ser útil para lidar com uma grande quantidade de dados. Se seleccionar esta definição, a impressora envia o documento para o spool do disco rígido e depois imprime-o a partir da fila de espera do disco, reduzindo a carga no computador.

Programa de impressão

Seleccione esta definição para imprimir o documento a uma hora especificada. Se definir a hora para as 23h quando a impressora está a ser pouco utilizada, a impressora guarda o documento no disco rígido e imprime-o às 23h.



As opções de Propriedades do trabalho estão disponíveis como se segue.

ID utilizad.

Introduza o ID do utilizador. Pode ser utilizado quando imprime o ficheiro mais tarde ou quando encontra o ficheiro guardado utilizando o painel de controlo da impressora.

Nome do trab.

Introduza o nome do trabalho de impressão. Pode ser utilizado quando imprime o ficheiro mais tarde ou quando encontra o ficheiro guardado utilizando o painel de controlo da impressora.

Palavra-passe

Introduza a palavra-passe que será utilizada para a definição Confidencial. Esta opção apenas está disponível quando selecciona Confidencial ou Armazenar sem assinalar Armazenar em HDD público. Se definiu a palavra-passe para um ficheiro específico, tem de introduzi-la para imprimir ou eliminar o ficheiro.

Imprimir em

Seleccione esta opção para definir a hora quando a impressora começa a impressão. Apenas pode utilizar esta definição se seleccionou Diferimento.
