Options avancées


Filigrane

La liste déroulante Filigrane vous permet d’activer ou de désactiver la fonction Filigrane.

    CONFIDENTIAL

    Permet de sélectionner le filigrane CONFIDENTIAL.

    DRAFT

    Permet de sélectionner le filigrane DRAFT.

    TOP SECRET

    Permet de sélectionner le filigrane TOP SECRET.

    Perso

    Permet de saisir le filigrane à imprimer.

    Options avancées

    Permet de définir les informations relatives à la police.

      Nom : permet de sélectionner le nom de la police.

      Style de texte : permet de sélectionner le style de la police.

      Couleur : permet de sélectionner la couleur de la police.

      Taille : permet de sélectionner la taille de la police (entre 8 et 400).

      Ombre : permet de sélectionner le contraste.

      Inclinaison : permet de modifier l’angle d’inclinaison du filigrane entre -90 et +90 degrés.

      Première page seulement : imprime le filigrane sur la première page seulement. Par défaut, cette case n’est pas cochée.



Affiche

Affiche : vous permet d’imprimer un document de grande taille sur plusieurs feuilles.

    ▪ Mise en page : permet de choisir le nombre de pages à utiliser lorsque vous imprimez une affiche. Par exemple, la valeur Affiche 2x2 imprime le document sur 2 pages en longueur et 2 pages en largeur, soit un total de 4 pages.



Brochure

Impression brochure : vous permet de choisir le type de brochure que vous souhaitez réaliser.

    Méthode de pliage

      ▪ Toutes les pages en même temps : vous permet de créer un document relié avec toutes les pages.

      ▪ Divisées en jeux : vous permet d’indiquer le nombre de feuilles dans chaque jeu lorsque vous imprimez un document relié dans ce mode. Vous pouvez indiquer le nombre de feuilles par jeu dans [Feuilles par jeu].

    Options de pliage : vous permet d’indiquer le mode de pliage du document relié imprimé.

      ▪ Sans : vous permet de créer un document relié avec toutes les pages.

      ▪ Plier : sélectionnez cette option pour que le document relié soit plié le long de la ligne du milieu du papier.

      ▪ Plier et agrafer : sélectionnez cette option pour utiliser à la fois la fonction de pliage et d’agrafage.

    Reliure : vous permet d’indiquer la largeur de l’espace entre la zone imprimée et la reliure devant rester vierge lors de l’impression d’un document relié.

    Reliure droite : sélectionnez cette option pour créer un document relié de droite à gauche.



Brochure manuelle


Impression brochure : vous permet de choisir le type de brochure que vous souhaitez réaliser.

La valeur par défaut est Désactivé.




REMARQUE :

Reliure de page n’est pas pris en charge dans Brochure manuelle.

Reliure droite n’est pas pris en charge dans Brochure manuelle.

Pour une Brochure manuelle, lorsque l’imprimante s’arrête, suivez les indications à l’écran.

Cette option peut ne pas être disponible selon le format et le type de papier sélectionnés.

Dans Mise en page, si le Nombre de pages par feuille est supérieur à 1, les paramètres Affiche, Brochure et Brochure manuelle seront définis sur Désactivé (ou sur Sans) et seront automatiquement désactivés.

Si le nombre de copies est supérieur à 1, le paramètre Brochure manuelle sera défini sur Désactivé et sera automatiquement désactivé.

Ne pas utiliser Recto-verso manuel avec Page de couverture.

Ne pas utiliser Brochure manuelle avec Page de couverture.
