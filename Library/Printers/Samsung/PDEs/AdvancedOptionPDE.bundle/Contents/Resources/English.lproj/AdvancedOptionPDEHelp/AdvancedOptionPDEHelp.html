Advanced Options


Watermark

The Watermark drop-down list allows the user to enable or disable the watermark feature.

    CONFIDENTIAL

    Allows users to select CONFIDENTIAL watermark.

    DRAFT

    Allows users to select DRAFT watermark.

    TOP SECRET

    Allows users to select TOP SECRET watermark.

    Custom

    Allows users to enter the watermark which is the user want to print.

    Advanced Options

    Allows users to set the font details.

      Name : Allows users to select the font name.

      Text Style : Allows users to select the font style.

      Color : Allows users to select the font color.

      Size : Allows users to select the font size from 8 to 400 pitch.

      Shade : Allows users to select the darkness.

      Message Angle : Allows users to change the watermark angle from -90 to +90 degrees.

      First Page Only : Prints watermark only on the first page. Not checked by default.



Poster

Poster Printing allows you to print large, multiple-sheet versions of your page.

    ▪ Pages Layout allows you to choose how many pages should be printed when you print a poster printing, For instance, Poster 2x2 will result in 2 pages long and 2 pages wide for a total of 4 pages



Booklet

Booklet Printing allows you to you to choose the kind of the booklet you want to produce.

    Fold Method

      ▪ All Pages At Once allows you to create a booklet with all pages.

      ▪ Divide Into Sets allows you to specify the number of sheets in each set when printing booklet in Divide Into Sets. You can specify the number of sheets per set in [Sheets per Set].

    Folding Options spcify how you fold the output of the bookelet printing.

      ▪ None : allows you to create a booklet with all pages.

      ▪ Fold : Select this option to fold a booklet along the middle line of the paper.

      ▪ Fold and Staple : Select this option to use both Staple and Folding feature.

    Gutter allows you to specify the width of the space between the printed area and binding to be left unprinted when booklet printing.

    Right Binding : Select this option to create a right-to-left booklet.



Manual Booklet


Booklet Printing allows you to choose the kind of the booklet you want to produce.

The default is Off.




NOTE:

Page Gutter is not supported in Manual Booklet.

Right Binding is not supported in Manual Booklet.

For Manual Booklet, when the printer stops printing, follow the steps as you can see in the picture on computer screen.

The option may not be available according to the selected paper size or type.

If number of Pages per Sheet in Layout is greater than 1, Poster, Booklet and Manual Booklet will be set to Off(or None) and disabled automatically.

If number of Copies is greater than 1, Manual Booklet will be set to Off and disabled automatically.

Do not use Manual Duplex with Cover Page.

Do not use Manual Booklet with Cover Page.
