Manual Duplex


The default is None.


▪None:

Printing occurs on one side of the page only.


▪Long Edge:

The document is printed so that it can be bound along the long edge of the page and read by turning the pages from left to right like a book. When the orientation of your document is set to landscape, and you choose Long Edge, the resulting output is a document that can be bound along the long edge and is read by flipping the pages from bottom to top, like a flipchart.


▪ Short Edge:

The document is printed so that it can be bound along the short edge of the paper and read by turning the pages from bottom to top, like a flipchart. When the orientation of your document is set to Landscape and you choose Short Edge, the resulting output is a document that can be bound along the short edge and is read by turning the pages from left to right like a book.


NOTE:

Reverse Duplex Printing is not supported in Double-sided Printing (Manual).

The option may not be available according to the selected paper size or type.

For Double-sided Printing (Manual), when the printer stops printing, follow the steps as you can see in the picture on computer screen.

If number of Copies is greater than 1, Manual Duplex will be set to None and disabled automatically.
