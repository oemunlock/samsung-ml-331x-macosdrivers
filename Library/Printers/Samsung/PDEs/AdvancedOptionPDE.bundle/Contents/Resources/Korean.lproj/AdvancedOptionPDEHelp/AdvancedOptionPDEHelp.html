고급설정


워터마크

워터마크는 사용자가 워터마크 기능을 선택 할 수 있도록 합니다.

    CONFIDENTIAL

    사용자가 CONFIDENTIAL워터마크를 사용 할 수 있도록 합니다.

    DRAFT

    사용자가 DRAFT워터마크를 사용 할 수 있도록 합니다.

    TOP SECRET

    사용자가 TOP SECRET워터마크를 사용 할 수 있도록 합니다.

    사용자 정의

    사용자가 직접 워터마크 메시지를 입력 할 수 있도록 합니다.

    고급 설정

    글꼴을 상세하게 설정하도록 합니다.

      이름 : 글꼴의 이름을 선택하도록 합니다.

      글꼴스타일 : 글꼴의 스타일을 선택하도록 합니다.

      색상 : 글꼴의 색상을 선택하도록 합니다.

      크기 : 글꼴의 크기를 8 부터 400 피치까지 선택하도록 합니다.

      명암 : 글꼴의 명암을 선택하도록 합니다.

      메세지 각도 : 워터마크의 각도를 -90도 부터 +90도 까지 변경하도록 합니다.

      첫 페이지만 인쇄 : 첫 페이지만 워터마크를 인쇄합니다. 기본설정은 체크하지 않도록 되어있습니다.



포스터

포스터 형식 인쇄 는 인쇄물을 확대해서 한 페이지 인쇄물을 여러 장으로 인쇄하도록 합니다.

    ▪ 페이지 레이아웃 은 포스터 인쇄 시 몇 장으로 할 것인가를 선택하도록 합니다 . 예를 들면 포스터<2x2>인 경우 1장을 인쇄하면 가로 세로가 각각 확대되어 4장이 인쇄됩니다.




책 형식 인쇄

책 형식 인쇄인쇄는 인쇄물의 다양한 책형식을 선택하도록 합니다.

    접기 방법

      ▪ 한 번에 접기 접기 는 모든 페이지를 한 번에 접어서 소책자를 완성하도록 출력합니다.

      ▪ 여러 부로 나누어 접기 는 한 번에 접을 페이지 수를 지정하여 세트단위로 나누어 출력합니다. 한 번에 접을 페이지 수는 [한 부 당 종이 수] 를 통해 설정 가능합니다.

    접기 설정설정 을 통해 소책자 출력물을 접는 방법을 선택합니다.

      ▪ 없음 : 소책자를 접는 방법을 지정하지 않습니다.

      ▪ 접기 : 출력물의 가운데 반을 기준으로 접어서 소책자를 완성합니다.

      ▪ 스테이플하고 접기 : 출력물의 가운데 반을 기준으로 스테이플을 찍고 접어서 소책자를 완성합니다.

    접기 여백 은 소책자 출력물을 접을 경우를 고려하여, 인쇄 영역과 접는 선간의 여백을 설정할 수 있도록 합니다.

    책장을 오른쪽에서 왼쪽으로 넘기기 : 오른쪽에서 왼쪽으로 책장을 넘겨 볼수 있는 소책자를 인쇄합니다.


책 형식 인쇄 (수동)


책 형식 인쇄는 인쇄물을 책형식으로 인쇄하도록 합니다.

기본값은 없음입니다.




참고:

책 형식 인쇄 (수동)인 경우, 접기여백 기능은 지원하지 않습니다.

책 형식 인쇄 (수동)인 경우, 책장을 오른쪽에서 왼쪽으로 넘기기 기능은 지원하지 않습니다.

책 형식 인쇄 (수동)인 경우, 프린터가 인쇄를 멈추면 인쇄 도움말 화면에 보이는대로 해 주십시오.

이 기능은 선택된 용지 크기 또는 용지 종류에 따라서 지원되지 않을 수도 있습니다.

레이아웃의 용지당 페이지 수가 1 이상이면, 포스터, 책 형식 인쇄, 책 형식 인쇄(수동)이 자동으로 꺼지고 비활성화 됩니다.

용지 매수가 1보다 크면, 책 형식 인쇄 (수동)이 자동으로 꺼지고 비활성화 됩니다.

표지 페이지에는 수동 양면을 사용하지 마십시오.

표지 페이지에는 수동 책 형식을 사용하지 마십시오.
